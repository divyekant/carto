# Social Posts: CLI Overhaul (v1.2.0)

## Twitter / X (under 280 characters)

Most CLIs need a --json flag for machine output. Carto auto-detects human vs. AI agent and adapts. No flags. No config. Same command, two output modes. The first agent-native CLI for codebase intelligence. v1.2.0 is out now.

## LinkedIn

We just shipped Carto v1.2.0 -- and it changes how we think about CLI design.

AI agents now run CLI commands millions of times a day on behalf of developers. But every CLI they encounter was built for human eyes: colored text, spinners, natural-language errors.

Carto v1.2.0 solves this at the design level. Every command auto-detects its consumer. Humans get rich terminal output. AI agents get structured JSON with typed error codes. No flags, no configuration, no parsing gymnastics.

We also added 9 new commands (18 total), shell completions, data export/import, structured audit logging, and self-update. Zero breaking changes.

The era of CLIs built only for humans is ending. Tools need to serve both the developer and the AI agent working alongside them.
